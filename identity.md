# Идентификация системы

## Название
**AI Content Pipeline v1.0** — Мультиагентная система генерации и публикации контента

## Версия
1.0.0 (Production-Ready)

## Идентификационные данные системы

| Параметр | Значение |
|----------|----------|
| Timezone | Europe/Samara (UTC+4) |
| Execution Window | 09:30 – 11:00 (ежедневно) |
| Orchestrator | Cron / Scheduler Agent |
| Storage | Google Sheets + Local FS (data/) |
| Output Channel | Telegram Bot API |

## Основная цель

Автоматизированный конвейер сбора новостей об ИИ → выделение трендов → генерация провокационных постов → создание обложек → публикация в Telegram-канал с минимальным участием человека.

## Принципы работы

| Принцип | Описание |
|---------|----------|
| **Ясность > Скорость** | Каждый агент имеет чёткие входы/выходы |
| **Измеримость** | Все этапы логируются в Google Sheets |
| **Безопасность** | Никаких секретов в коде, токены через .env |
| **Идемпотентность** | Повторный запуск не дублирует данные |
| **Graceful Degradation** | При сбое одного агента — логирование, не остановка всего пайплайна |

## Тон

Спокойный, аналитический, структурированный. Без драматизации.
